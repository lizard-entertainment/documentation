VerticalFitter
This script is used to resize Portraits so their holder can dynamically space them out when buffs are added underneath them. Adding a vertical layout group to each portrait does not work because we need custom anchors on all the child gameobjects of the portrait so the game can rescale to various aspect ratios, and a vertical layout group prevents this by forcing a particular alignment on its children.
