ObjectPoolManager
This class registers objects and retrieves them when needed, greatly improving the performance of the game as we only call Instantiate in the beginning, resetting and retrieving gameObjects should we need them again.
PooledBehaviour
Represents a MonoBehavior that can be pooled. Pooled objects are Reset() and returned to the object pool when "destroyed." So we can retrieve them from there instead of doing costly instantiations.
